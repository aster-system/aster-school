Aster School

Warning : As the lessons come from France, the entire website is in french.

Présentation

Aster School est une branche de l'organisation Aster Système.
Il s'agit d'un site web, regroupant pleins de cours de terminal pris par le créateur d'Aster Système, Mattéo, et d'autres membres.
Cependant, les cours seront re-travaillés pour être facilement compréhensible par n'importe qui.
De plus, certains cours du programme peuvent paraître assez peu poussé et peuvent être approfondis. C'est pour ça que les cours seront de temps en temps améliorés avec des éléments hors programmes.

Pourquoi Aster School ?

Tous les cours présents sur Aster School sont rédigés par des personnes qui suivent également ces cours. Ils sont donc les plus adéquat du monde pour proposer ce service.

Réalisation

Ce site web est réalise avec les très cèlèbres langages HTML et CSS, vu en SNT (même si le niveau nécessaire pour créer un site de ce genre est bien supérieur à ce que vous avez vu).
Pour la structuration, il utilise l'outil de l'organisation mère, Aster Système, nommé SCLS Documentalist Agatha.

La version actuelle du site web, modifiée le 20/04/2024, n'ai pas encore équipée de style, ni de structure propre. Cependant, ça ne tardera pas.
